As part of our efforts to stay in tune to our users and provide the best functionality possible for all, we have completed the Message Board Enhancement.  Please read below for any actions that you will need to take to complete this migration.

Enhanced Message Board Interface

The new Message Board interface that went live on Feb 7, 2009 includes a number of features that users have been requesting.  The Message Boards (Forums) were enhanced to integrate tighter with the social networking components that are being introduced into the community.  Please take some time to login and look at the new interface.

We have also implemented easier methods to get notifications on new message threads.  As part of this migration process, each user will want to access the forums to update their email subscriptions (re-subscribe).
